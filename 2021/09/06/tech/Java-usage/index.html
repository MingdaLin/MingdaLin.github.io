自动装箱&拆箱

  • 什么是自动装箱&拆箱？

Java中基础数据类型与它们的包装类进行运算时，编译器会自动帮我们进行转换，转换过程对程序员是透明的，这就是装箱和拆箱，

  • 什么时候会进行装箱&拆箱？
    • 进行 = 赋值操作（装箱或拆箱）
    • 进行+，-，*，/混合运算 （拆箱）
    • 进行>,<,==比较运算（拆箱）
    • 调用equals进行比较（装箱）
    • ArrayList,HashMap等集合类 添加基础类型数据时（装箱）
